Telegram-бот для оценки показателей здоровья на основе экспертной модели
Данный проект представляет собой Telegram-бота, реализующего экспертную систему для анализа базовых показателей здоровья пользователя.
Проект был разработан в рамках магистерской дипломной работы. В ходе работы были проанализированы рекомендации и открытые данные Всемирной организации здравоохранения (ВОЗ)
, на основе которых была сформирована экспертная модель оценки показателей здоровья.
На основании анализа данных ВОЗ были определены пороговые значения различных показателей, которые используются системой для оценки состояния пользователя.
Telegram-бот взаимодействует с пользователем, собирает необходимые данные и сравнивает их с установленными пороговыми уровнями, после чего формирует рекомендации или предупреждения.

Основные этапы проекта
Анализ рекомендаций и статистических данных Всемирной организации здравоохранения (ВОЗ)
Формирование экспертной модели для оценки показателей здоровья
Создание файлов с пороговыми значениями для различных параметров
Разработка Telegram-бота на Python с использованием библиотеки telebot (pyTelegramBotAPI)
Реализация логики взаимодействия с пользователем через Telegram
Сравнение введённых пользователем данных с установленными пороговыми значениями и формирование рекомендаций

Используемые технологии
Python
telebot (pyTelegramBotAPI)

Цель проекта — разработка экспертной системы, которая способна анализировать пользовательские данные и предоставлять рекомендации на основе заранее определённых правил и медицинских норм.
